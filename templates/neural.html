The regression model let users easily see which variables have the biggest impact. This can be done by simply comparing its coefficients in the fitting function. However, the neural network has no such coefficients since there is no fitting function to begin with. For the neural network, we determine such variables by comparing the change in R value. Below is the list of variables that are found to be important.

  1. Age
  2. Number of product
  3. Active member
  4. Germany
  5. Gender
  6. Balance
  7. Credit score

We see the biggest change in R value when we include/exclude "Age" in the model indicating Age plays an important role, the same conclusion arrived in the regression model. The difference is the current neural network includes "Number of product" as the second biggest factor where as the regression model did not include this variable. We tested the regression model again with "Number of product" included, though it did not improve the result showing its non-linear behavior that regession model could not fit.
